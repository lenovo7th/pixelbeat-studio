# 🎵 PixelBeat Studio

A browser-based music sequencer and DAW designed for game developers.
Create, compose, and export game music directly in your browser.

## ✨ Features

- 🥁 Step Sequencer — 16-step grid with Drums, Piano, Guitar, Strings
- 🎤 Voice Studio — Professional vocal processing chain
- 🎛️ Effects Chain — Compressor, EQ, Reverb, Delay, Chorus, Distortion
- 🎼 Pattern System — Up to 4 patterns, chainable for game loops
- 🎚️ Real-time BPM control
- 🌊 Live visualizer — Waveform + FFT spectrum

## 🛠️ Built With

- Tone.js
- Web Audio API
- HTML / CSS / JavaScript

## 🎮 Made For

Game developers who need a quick, lightweight tool
to compose music and process audio without leaving the browser.
